#### 一、申请百度AK

> 不管原生，还是Vue还是React项目，不使用插件引入：在script中引入
>
> ```react
> <script  type="text/javascript" src="http://api.map.baidu.com/api?v=2.0&ak=xxxxxxxx" ></script>
> ```
>
> 项目开始是会自动将BMap注入到window原型里，全局使用
